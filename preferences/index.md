## [Claude.ai と Claude Code](#claude-ai-and-claude-code) {#claude-ai-and-claude-code}

Claudeには2つの使い方があって、それぞれ覚え方が違います

|                              |                                       |
| ---------------------------- | ------------------------------------- |
| [Claude.ai](./claude-ai)     | アプリで使うClaude（7つの覚え方）     |
| [Claude Code](./claude-code) | ターミナルで使うClaude（5つの覚え方） |
