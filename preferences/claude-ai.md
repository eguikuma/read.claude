# [これはなに？](#what-is-this) {#what-is-this}

アプリ版のClaudeには、7つの覚え方があります

ここでは、その覚え方を紹介します

---

## [エンタープライズ](#enterprise) {#enterprise}

### [Enterprise Memory Controls](#enterprise-memory-controls) {#enterprise-memory-controls}

組織全体で使う設定をまとめられます

|            |                            |
| ---------- | -------------------------- |
| 範囲       | 組織の全員                 |
| どこで設定 | 管理コンソール             |
| 使いどころ | 組織のルールをみんなに反映 |

{: .source}
[公式ドキュメント](https://support.claude.com/en/articles/12622667-enterprise-configuration){:target="\_blank"}

---

## [プロジェクト](#project) {#project}

### [Project Instructions](#project-instructions) {#project-instructions}

そのプロジェクト専用の指示を書く場所です

例：「このプロジェクトではXXXパターンを使用して...」

|            |                                      |
| ---------- | ------------------------------------ |
| 範囲       | そのプロジェクトだけ                 |
| どこで設定 | プロジェクト → 指示                  |
| 使いどころ | そのプロジェクトだけのルールを決める |

{: .source}
[公式ドキュメント](https://support.claude.com/en/articles/9519177-how-can-i-create-and-manage-projects){:target="\_blank"}

---

### [Project Memory And Summary](#project-memory-and-summary) {#project-memory-and-summary}

プロジェクト内の会話からClaudeが覚えてくれた内容です

話せば話すほど、そのプロジェクトのことを分かってくれます

|            |                                |
| ---------- | ------------------------------ |
| 範囲       | そのプロジェクトだけ           |
| どこで設定 | 自動でたまっていく             |
| 使いどころ | プロジェクトの流れを覚えておく |

{: .source}
[公式ドキュメント](https://support.claude.com/en/articles/11817273-using-claude-s-chat-search-and-memory-to-build-on-previous-context){:target="\_blank"}

---

## [ユーザー](#user) {#user}

### [Profile Preferences](#profile-preferences) {#profile-preferences}

あなたのことを教えてあげる場所です

例：「私はフロントエンドエンジニアで、TypeScriptをよく使います」

|            |                      |
| ---------- | -------------------- |
| 範囲       | すべての会話         |
| どこで設定 | 設定 → プロフィール  |
| 使いどころ | 名前、職業、興味など |

{: .source}
[公式ドキュメント](https://support.claude.com/en/articles/10185728-understanding-claude-s-personalization-features){:target="\_blank"}

---

### [Style](#style) {#style}

Claudeの答え方をカスタマイズできます

複数のスタイルを作って、場面に応じて切り替えられます

|            |                                      |
| ---------- | ------------------------------------ |
| 範囲       | すべての会話（切り替えOK）           |
| どこで設定 | 設定 → スタイル                      |
| 使いどころ | 回答の長さ、トーン、フォーマットなど |

{: .source}
[公式ドキュメント](https://support.claude.com/en/articles/10181068-configuring-and-using-styles){:target="\_blank"}

---

### [Memory And Summary](#memory-and-summary) {#memory-and-summary}

ふだんの会話から覚えてくれた内容です

話すほど、あなたのことを分かってくれるようになります

|            |                              |
| ---------- | ---------------------------- |
| 範囲       | プロジェクト外のすべての会話 |
| どこで設定 | 自動でたまっていく           |
| 使いどころ | あなたの好みを覚える         |

{: .source}
[公式ドキュメント](https://support.claude.com/en/articles/11817273-using-claude-s-chat-search-and-memory-to-build-on-previous-context){:target="\_blank"}

---

### [Incognito Chats](#incognito-chats) {#incognito-chats}

覚えないチャットモードです

このモードでの会話は覚えてもらえず、履歴にも残りません

|            |                                  |
| ---------- | -------------------------------- |
| 範囲       | そのチャットだけ                 |
| どこで設定 | チャット作成時に選ぶ             |
| 使いどころ | 一時的な質問、プライベートな内容 |

{: .source}
[公式ドキュメント](https://support.claude.com/en/articles/12260368-using-incognito-chats){:target="\_blank"}
